Bring open web technologies to your browser

Google Chrome Frame is a free plug-in for Internet Explorer. Some advanced web apps, like Google Wave, use Google Chrome Frame to provide you with additional features and better performance.

Note: This is an early developer preview of Google Chrome Frame.

Get Google Chrome Frame

It is free and it installs in seconds.

For Internet Explorer 6, 7 and 8 on Windows 7 / Vista / XP SP2

You will need administrative rights on your computer to install Google Chrome Frame

Websites ask you to download Google Chrome Frame, to:

Have a faster experience
Sites that utilize Google Chrome Frame become more responsive.

Enjoy more advanced features
Open technologies in Google Chrome Frame let sites create new features for users.

Surf the web the way you are used to
Google Chrome Frame works under the hood with your current browser.
